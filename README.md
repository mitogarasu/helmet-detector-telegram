# Обнаружение людей без защитных шлемов с оповещениями в Telegram

Система на основе YOLOv5, которая обнаруживает людей без защитных шлемов на видео и отправляет оповещения со скриншотами боту в Telegram.

## Функции
- Обнаружение людей/шлемов
- Красная рамка для людей без шлемов
- Уведомления Telegram с отображением людей без шлема
